# Docs 目录索引

当前 `docs` 已按主题整理为以下目录：

- `接口文档/`
  - 通用接口说明、Swagger 示例、接口测试相关文档
- `任务与调度/`
  - 任务编辑、定时任务、调度能力相关文档
  - `任务与调度/多品数据回测维护说明.md`
    - 多品数据回测的入口、配置结构、执行链路、预览比例计算和后续修改检查清单
  - `任务与调度/多品数据回测比例算法审查.md`
    - 多品比例的指标层加权现状、真实组合算法差异、受影响指标和修正建议
- `前端/`
  - 前端维护、Vue 迁移、页面对齐、UI 方案相关文档
  - `前端/前端架构总览.md`
    - Vue 3 前端技术栈、目录结构、路由架构、API 层、主题系统、响应式策略、性能优化
  - `前端/组件库文档.md`
    - 11 个可复用组件的完整 API 文档（Props / Events / Slots / 使用示例）
  - `前端/Composables文档.md`
    - 7 个组合式函数的 API 文档（usePolling / useResponsive / useDebounce 等）
- `后端/`
  - 后端维护、鉴权、权限架构、后端优化相关文档
- `数据库/`
  - 数据库优化与参考文档
- `日志与监控/`
  - 日志接口、日志优化、异常与变更记录
- `项目规划/`
  - 项目分析、评审报告、专项方案文档
  - `项目规划/平台演进蓝图-2026-05-15/`
    - 基于当前平台上下文的架构演进方案
    - 多 AI Agent 并行开发协作方案
- `服务器接口映射/`
  - 面向外部服务联调的接口映射与输入输出说明

联调文档入口：

- `服务器接口映射/任务接口对接说明.md`
