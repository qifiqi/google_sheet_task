# 前端对齐方案

> 目标：新 Vue 前端与老 Jinja2 前端功能一一对齐，组件换用 Element Plus，字体/间距调大。
> 生成日期：2026-04-07

---

## 一、全局字体与样式调整

**问题：** Element Plus 默认字体 14px，老前端 Bootstrap 基准 16px，视觉偏小。

**修改文件：** `frontend/src/styles/index.scss`

在现有内容基础上追加：

```scss
// 覆盖 Element Plus 全局字体变量
:root {
  --el-font-size-base:        15px;   // 默认 14px
  --el-font-size-small:       13px;   // 默认 12px
  --el-font-size-extra-small: 12px;   // 默认 10px
  --el-font-size-medium:      16px;   // 默认 14px
  --el-font-size-large:       18px;   // 默认 16px

  --el-table-row-height:      44px;   // 默认 40px，行更宽松
  --el-component-size:        36px;   // 输入框/按钮高度，默认 32px
  --el-component-size-small:  28px;   // 小号组件，默认 24px
}

// 页面标题统一大小
h2 { font-size: 18px; }
h3 { font-size: 16px; }

// 表格内文字
.el-table { font-size: 14px; }

// 卡片统计数字
.stat-value { font-size: 24px; font-weight: 600; }
.stat-label { font-size: 13px; color: #909399; }
```

---

## 二、组件使用规范

| 场景 | 组件 | 规格 |
|------|------|------|
| 数据表格 | `el-table` | 不传 size（使用全局 44px 行高） |
| 搜索/筛选输入 | `el-input` / `el-select` | `size="default"` |
| 表格内操作按钮 | `el-button` | `size="small"` |
| 页面级操作按钮 | `el-button` | `size="default"` |
| 弹窗 | `el-dialog` | PC 宽 600px，移动端 `width="95%"` |
| 分页 | `el-pagination` | `layout="total, sizes, prev, pager, next"` |
| 状态标签 | `el-tag` | `size="small"` |
| 统计卡片数字 | `el-statistic` 或自定义 div | 数字 24px，标签 13px |
| 详情抽屉 | `el-drawer` | `size="600px"`（PC），移动端 `size="100%"` |
| 表单标签 | `el-form` | `label-width="100px"`，移动端 `label-position="top"` |
