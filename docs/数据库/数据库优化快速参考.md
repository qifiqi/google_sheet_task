# 🚀 数据库优化快速参考卡

## 一键优化

```bash
# 最简单的方式 - 一键应用所有优化
python apply_db_optimizations.py
```

## 分步执行

### 1️⃣ 备份数据库
```bash
# Windows
copy instance\app.db instance\app.db.backup

# Linux/Mac
cp instance/app.db instance/app.db.backup
```

### 2️⃣ 应用迁移（添加索引）
```bash
flask db migrate -m "添加索引优化"
flask db upgrade
```

### 3️⃣ 清理旧日志
```bash
python cleanup_task_logs.py
```

### 4️⃣ 压缩数据库
```bash
python db_performance_check.py --vacuum
```

### 5️⃣ 性能检查
```bash
python db_performance_check.py --full
```

## 代码示例

### 批量插入
```python
from app.utils.db_optimizer import batch_insert
from app.models import TaskResult

batch_insert(TaskResult, data_list, batch_size=100)
```

### 批量更新
```python
from app.utils.db_optimizer import batch_update

updates = [{'id': 'task1', 'status': 'completed'}, ...]
batch_update(Task, updates, batch_size=100)
```

### 批量删除
```python
from app.utils.db_optimizer import batch_delete

batch_delete(TaskLog, old_ids, batch_size=500)
```

### 避免 N+1
```python
from app.utils.db_optimizer import QueryOptimizer

query = QueryOptimizer.prefetch_related(Task.query, Task.results)
tasks = query.all()
```

### 高效分页
```python
from app.utils.db_optimizer import query_with_pagination

items, total, pages = query_with_pagination(query, page=1, per_page=20)
```

### 性能监控
```python
from app.utils.db_monitor import DatabaseMonitor

report = DatabaseMonitor.get_full_report()
print(f"数据库: {report['database']['size_mb']} MB")
```

## 故障排除

### 迁移失败
```bash
flask db downgrade  # 回滚
flask db upgrade    # 重新应用
```

### 数据库锁定
```bash
# 检查连接池
python -c "from app.utils.db_monitor import DatabaseMonitor; \
from app import create_app; \
app = create_app(); \
with app.app_context(): \
    print(DatabaseMonitor.get_connection_pool_status())"
```
