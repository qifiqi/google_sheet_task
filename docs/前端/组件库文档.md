# Vue 前端组件库文档

> 本文档描述 `frontend/src/components/` 下所有可复用组件的 API、使用方式和设计约定。

## 组件总览

| 组件 | 文件 | 用途 |
|------|------|------|
| StatusTag | `StatusTag.vue` | 任务状态标签 |
| PageToolbar | `PageToolbar.vue` | 页面标题区 |
| StatCardGrid | `StatCardGrid.vue` | 统计指标卡片网格 |
| TaskProgressCell | `TaskProgressCell.vue` | 表格进度条单元格 |
| CodeBlock | `CodeBlock.vue` | JSON/代码预格式化显示 |
| EmptyState | `EmptyState.vue` | 空状态占位 |
| FilterToolbar | `FilterToolbar.vue` | 通用筛选栏 |
| DataTableCard | `DataTableCard.vue` | 卡片表格 + 分页 |
| LogViewer | `LogViewer.vue` | 终端风格日志查看器 |
| ChartPanel | `ChartPanel.vue` | Chart.js 画布容器 |
| MobileCardList | `MobileCardList.vue` | 移动端卡片列表 |

## PageToolbar

每个页面顶部的标题区域，统一 eyebrow + title + description + 操作按钮布局。

```vue
<PageToolbar
  eyebrow="Task Queue"
  title="C3 任务列表"
  description="查看任务排队、执行进度和错误状态。"
>
  <template #actions>
    <el-button type="primary">创建新任务</el-button>
  </template>
</PageToolbar>
```

| Prop | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| eyebrow | string | `''` | 标题上方的小标签 |
| title | string | **必填** | 页面主标题 |
| description | string | `''` | 副标题描述 |

| Slot | 说明 |
|------|------|
| #actions | 右侧操作按钮区 |

## StatCardGrid

统计指标卡片网格，支持 gradient 和 outlined 两种风格。

```vue
<StatCardGrid
  :cards="[
    { key: 'total', label: '总任务数', hint: '全部历史任务' },
    { key: 'running', label: '运行中', background: 'linear-gradient(135deg, #d97706, #f59e0b)' },
  ]"
  :data="{ total: 128, running: 3 }"
  :columns="{ xs: 12, sm: 6, md: 4 }"
  variant="gradient"
/>
```

| Prop | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| cards | Array | **必填** | 卡片定义 `{ key, label, hint?, background?, color? }` |
| data | Object | `{}` | 数据对象，key 对应 cards 中的 key |
| columns | Object | `{ xs: 12, sm: 6, md: 6 }` | 响应式列宽 |
| variant | string | `''` | 风格：`'gradient'` 白色文字 / 默认边框卡片 |

| Slot | 说明 |
|------|------|
| #card({ card, value }) | 自定义单个卡片内容 |

## FilterToolbar

通用筛选栏，支持 select / input / date 三种控件类型。input 类型自带 300ms 防抖。

```vue
<FilterToolbar
  v-model="filters"
  :filters="[
    { key: 'status', type: 'select', placeholder: '状态', options: [...], span: { xs: 24, sm: 6 } },
    { key: 'keyword', type: 'input', placeholder: '搜索', span: { xs: 24, sm: 8 } },
    { key: 'date', type: 'date', placeholder: '日期' },
  ]"
  @search="handleSearch"
  @clear="handleClear"
>
  <template #extra>
    <el-button>导出</el-button>
  </template>
</FilterToolbar>
```

| Prop | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| filters | Array | **必填** | 筛选项定义 |
| modelValue | Object | `{}` | 当前筛选值 (v-model) |
| debounceDelay | number | `300` | input 防抖延迟 (ms) |

**Filter 定义结构：**
```ts
{
  key: string           // 字段名
  type: 'select' | 'input' | 'date'
  placeholder?: string
  label?: string
  options?: Array<{ value, label }>  // select 专用
  span?: { xs?, sm?, md? }           // 列宽
}
```

| Event | 说明 |
|-------|------|
| update:modelValue | 筛选值变化 |
| search | 用户触发搜索（回车/选择/防抖完成） |
| clear | 用户点击清空 |

| Slot | 说明 |
|------|------|
| #extra | 按钮区额外内容 |

## DataTableCard

卡片包裹的 el-table + 分页，移动端自动简化分页布局。

```vue
<DataTableCard
  title="任务列表"
  :loading="loading"
  :data="tasks"
  :total="total"
  v-model:page="page"
  v-model:page-size="pageSize"
  @page-change="loadTasks"
>
  <template #header-extra>共 {{ total }} 条</template>
  <el-table-column prop="name" label="名称" />
  <el-table-column prop="status" label="状态" />
</DataTableCard>
```

| Prop | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| title | string | `''` | 卡片标题 |
| loading | boolean | `false` | 加载状态 |
| data | Array | `[]` | 表格数据 |
| total | number | `0` | 总记录数 |
| page | number | `1` | 当前页 (v-model) |
| pageSize | number | `20` | 每页条数 (v-model) |
| pageSizes | Array | `[10, 20, 50]` | 可选页大小 |
| stripe | boolean | `true` | 斑马纹 |
| showPagination | boolean | `true` | 是否显示分页 |

| Event | 说明 |
|-------|------|
| update:page | 页码变化 |
| update:pageSize | 页大小变化 |
| page-change | 翻页/切换页大小后触发 |

| Slot | 说明 |
|------|------|
| #header-extra | 标题右侧附加信息 |
| #default | 表格列定义 |
| #empty | 空状态自定义 |

**响应式行为：** 移动端分页自动切换为 `prev, pager, next` 布局并居中。

## LogViewer

终端风格日志查看器，超过 500 条自动启用虚拟滚动。

```vue
<LogViewer
  :logs="logs"
  height="600px"
  :loading="loading"
  @scroll-top="loadMore"
/>
```

| Prop | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| logs | Array | `[]` | 日志数组 `{ timestamp?, level?, message, source? }` |
| height | string | `'500px'` | 容器高度 |
| autoScroll | boolean | `true` | 新日志自动滚动到底部 |
| loading | boolean | `false` | 加载状态 |
| virtualThreshold | number | `500` | 超过此数量启用虚拟滚动 |

| Event | 说明 |
|-------|------|
| scroll-top | 滚动到顶部（用于加载更多） |

| Slot | 说明 |
|------|------|
| #toolbar | 日志容器上方工具栏 |

**日志级别颜色：** info=蓝, warning=黄, error=红, debug=紫, success=绿

## TaskProgressCell

表格内进度条单元格，totalSteps 为 0 时显示 `-`。


| Prop | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| currentStep | number | `0` | 当前步骤 |
| totalSteps | number | `0` | 总步骤数 |

## CodeBlock

JSON/代码预格式化显示，超过 10KB 自动折叠。

```vue
<CodeBlock :content="task.config" max-height="400px" />
<CodeBlock :content="errorMessage" variant="danger" />
```

| Prop | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| content | string/Object/Array | `''` | 内容（对象自动 JSON.stringify） |
| maxHeight | string | `'300px'` | 最大高度 |
| variant | string | `'default'` | 风格：`'default'` / `'danger'` |

## StatusTag

任务状态标签，映射状态字符串到 Element Plus Tag 类型和中文标签。

```vue
<StatusTag status="running" />
```

| Prop | 类型 | 说明 |
|------|------|------|
| status | string | 状态值：pending/running/completed/cancelled/error |

**映射关系：**
- pending → info / 待执行
- running → warning / 运行中
- completed → success / 已完成
- cancelled → info / 已取消
- error → danger / 错误

## MobileCardList

移动端卡片列表，作为表格的移动端替代方案。

```vue
<MobileCardList :data="tasks" :loading="loading" @item-click="goDetail">
  <template #card="{ item }">
    <el-card shadow="hover">
      <div>{{ item.name }}</div>
      <StatusTag :status="item.status" />
    </el-card>
  </template>
</MobileCardList>
```

| Prop | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| data | Array | `[]` | 数据列表 |
| loading | boolean | `false` | 加载状态 |
| emptyText | string | `'暂无数据'` | 空状态文字 |

| Event | 说明 |
|-------|------|
| item-click(item) | 卡片点击 |

| Slot | 说明 |
|------|------|
| #card({ item, index }) | 单个卡片内容 |
