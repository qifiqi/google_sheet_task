## 🎯 使用示例

### 示例 1：批量插入结果

**优化前**：
```python
# ❌ 慢方法 - 单条插入
for result_data in results:
    result = TaskResult(**result_data)
    db.session.add(result)
    db.session.commit()  # 15 秒插入 1000 条
```

**优化后**：
```python
# ✅ 快方法 - 批量插入
from app.utils.db_optimizer import batch_insert

batch_insert(TaskResult, results, batch_size=100)  # 0.5 秒插入 1000 条
```

### 示例 2：避免 N+1 查询

**优化前**：
```python
# ❌ N+1 查询问题
tasks = Task.query.all()
for task in tasks:
    results = task.results.all()  # 每个任务一次查询
```

**优化后**：
```python
# ✅ 预加载关联数据
from app.utils.db_optimizer import QueryOptimizer

query = QueryOptimizer.prefetch_related(Task.query, Task.results)
tasks = query.all()
for task in tasks:
    results = task.results  # 无额外查询
```

### 示例 3：高效分页

**优化前**：
```python
# ❌ 低效方法
tasks = Task.query.all()
total = len(tasks)
page_tasks = tasks[(page-1)*20:page*20]
```

**优化后**：
```python
# ✅ 高效分页
from app.utils.db_optimizer import query_with_pagination

query = Task.query.filter_by(status='completed')
items, total, pages = query_with_pagination(query, page=1, per_page=20)
```

### 示例 4：监控数据库

```python
from app.utils.db_monitor import DatabaseMonitor

# 获取完整报告
report = DatabaseMonitor.get_full_report()

print(f"数据库大小: {report['database']['size_mb']} MB")
print(f"任务数量: {report['tables']['tasks']['count']}")
print(f"连接池状态: {report['connection_pool']['status']}")

# 获取优化建议
for suggestion in report['suggestions']:
    if suggestion['priority'] == 'high':
        print(f"⚠️  {suggestion['issue']}")
        print(f"   建议: {suggestion['suggestion']}")
```
